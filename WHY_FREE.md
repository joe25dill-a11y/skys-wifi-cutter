## How Netcut/Arcai Actually Works

They use the exact same methods this tool uses:

1. **ARP Table Reading**
   - Your OS maintains an ARP table
   - Both tools read this table
   - It's a standard OS feature
   - FREE to access

2. **Firewall Rules**
   - Your OS has a firewall (iptables/pfctl/Windows Firewall)
   - Both tools modify firewall rules
   - It's a standard OS feature
   - FREE to use

3. **Network Interface Stats**
   - Your OS tracks network traffic
   - Both tools read these statistics
   - It's a standard OS feature
   - FREE to monitor

**They charge you for accessing your own computer's features.**

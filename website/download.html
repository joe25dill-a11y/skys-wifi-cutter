Install steps

  1. 1. Download Skys.WiFi.Cutter.Setup.exe from the button above.
  2. 2. Right-click → Run as administrator (required for cut, lag, hotspot).
  3. 3. Allow Npcap if prompted; reboot once if scan/cut fails after first install.
  4. 4. Launch from Start Menu — complete the setup wizard, then Scan.
  5. 5. Optional: enable phone remote in Tools → Remote, then open http://<PC-IP>:3001/remote on your phone (same WiFi).

Features (v4.6.x)

  • Device scan & manufacturer lookup
  • Cut / restore / cut all + network defense
  • Lag switch, speed limits, ghost pulse
  • Per-device bandwidth metering
  • Hotspot freeze & gaming mode
  • DNS & port blocks + game presets
  • Schedules, groups, automation rules
  • Mobile remote page (opt-in PIN)

Having issues?

  • Run as Administrator for cut/lag/hotspot
  • Reboot once after first install if scan fails
  • Guest WiFi / AP isolation can block ARP cut — use the cut troubleshooting wizard in Tools
  • In app: Tools → Diagnostics → Copy feedback report, then paste into a GitHub issue

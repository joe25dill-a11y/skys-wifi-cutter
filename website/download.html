Install steps

  1. Right-click installer → Run as administrator
  2. Allow Npcap if prompted
  3. Launch from Start Menu
  4. Complete setup wizard → Scan

Features

  • Device scan & manufacturer lookup
  • Cut / restore / cut all
  • Lag switch & speed limits
  • Per-device bandwidth
  • Hotspot freeze & gaming mode
  • DNS & port blocks

Having issues?

  • Run as Administrator for cut/lag/hotspot
  • Reboot once after first install if scan fails
  • In app: Tools → Diagnostics → Copy feedback report
